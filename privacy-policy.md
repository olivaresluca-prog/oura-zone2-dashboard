# Privacy Policy

This application is a personal, local-use dashboard created to analyze Oura Ring data for private fitness and training purposes.

The application does not sell, share, or transfer user data to third parties.

Oura data is accessed only after user authorization through Oura OAuth and is used only to display personal metrics such as heart rate, workouts, readiness, sleep, and activity data.

Data is processed locally by the user. No commercial use is intended.

For questions, contact the application owner.
